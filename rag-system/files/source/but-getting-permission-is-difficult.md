# But getting permission is difficult!

**Question:** What should I do if getting permission to use a work seems too hard?


**Answer:** When selecting material to reuse, choosing content that is straightforward to clear for your intended use will save time. CLNZ can sometimes assist with one-off transactional licences for specific situations.
