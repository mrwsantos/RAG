# If a work is unpublished, does copyright apply?

**Question:** Does copyright apply to unpublished works?


**Answer:** Yes. Copyright applies as soon as a work is recorded in some way, regardless of whether it has been published. Saving a document to a computer file or writing on paper is sufficient. Formal publication is not required for copyright to arise.
