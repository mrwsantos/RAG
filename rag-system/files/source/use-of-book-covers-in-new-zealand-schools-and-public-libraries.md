# Use of book covers in New Zealand schools and public libraries

**Question:** Can a school or public library reproduce book cover images?


**Answer:** A joint statement from PANZ, NZSA, CLNZ, LIANZA, and SLANZA provides guidance for New Zealand schools and public libraries on this question. Prescribed libraries as defined in the Copyright Act may reproduce book cover images for non-commercial purposes without seeking specific permission from the publisher or author. This covers uses such as displays, posters, catalogues, social media posts, and websites, where the purpose is to promote the book or author. Uses that generate a financial return for the library, such as using a cover image on bookmarks sold to patrons, are not covered by this statement. When using a book cover, libraries should be mindful of the moral rights of the author, cover designer, and any other creators involved. Where practical, the cover should be used as a complete image rather than cropped in a way that compromises its integrity.
