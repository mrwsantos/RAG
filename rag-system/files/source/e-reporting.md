# E-reporting

**Question:** What is e-reporting and how does it work for CLNZ licenceholders?


**Answer:** E-reporting is an alternative to periodic data collection where licenceholders report copyright usage electronically throughout the year using specialised software or a learning management system plug-in. Data is extracted by the licenceholder and supplied to CLNZ regularly, several times a year, and must meet agreed criteria. There is no separate periodic data collection process under e-reporting. For e-reporting to work effectively, all material copied under the licence must be centralised within the software or plug-in so it is captured in reports. This approach also provides better oversight of course pack materials and is generally preferred by students.
