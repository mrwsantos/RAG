# Purposes of data collection

**Question:** Why does CLNZ ask licenceholders to carry out data collection?


**Answer:** CLNZ asks licenceholders to provide data on their copyright usage each year so that it can distribute licence fees to the rightholders (publishers and authors) whose material has been copied under the licence. Only a small proportion of licenceholders are asked to carry out data collection in any given year, and it is very unusual for the same licenceholder to be asked more than once in a five-year period. CLNZ uses a statistically representative sample to calculate usage across all licenceholders. Data collection is not treated as a compliance or enforcement exercise. If your data collection reveals that copying has exceeded the licence limits, no penalty is imposed, but you will need to seek permission from the rightsholders for the excess usage, as the licence cannot be relied on beyond its stated limits.
