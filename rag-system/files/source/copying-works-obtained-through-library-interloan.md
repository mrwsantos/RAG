# Copying works obtained through library interloan

**Question:** Can I copy a work obtained through library interloan?


**Answer:** You may copy works obtained through library interloan under CLNZ's trial library interloan licence extension. No application is required, but usage must be reported to the person at your organisation responsible for administering interloan requests, and that administrator must report usage to CLNZ. This service is currently available at universities and some ITPs and wānanga that have chosen to participate. If you work at an ITP, check whether your organisation is part of the trial before proceeding.
