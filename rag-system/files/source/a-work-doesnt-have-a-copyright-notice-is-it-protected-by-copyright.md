# A work doesn't have a copyright notice; is it protected by copyright?

**Question:** Does a work need a copyright notice to be protected by copyright?


**Answer:** Copyright protection in New Zealand is automatic from the moment a work is created and recorded in some way. No notice, registration, or © symbol is required. CLNZ recommends displaying a copyright notice because it signals to users that copyright exists and may deter infringement, but it has no bearing on whether protection applies.
