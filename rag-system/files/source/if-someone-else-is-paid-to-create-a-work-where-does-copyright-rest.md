# If someone else is paid to create a work, where does copyright rest?

**Question:** Who owns the copyright in work created by a paid contractor or employee?


**Answer:** It depends on whether the creator is an employee or an independent contractor. If the work is created by an employee in the course of their employment, copyright generally rests with the employer. If the creator is an independent contractor, they typically retain copyright unless there is a written agreement that transfers it. Where no written agreement exists, the commissioning party is usually entitled to use the work for the specific purpose it was commissioned for, but not for other purposes.
