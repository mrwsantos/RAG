# If changes are made to a work, is permission no longer needed?

**Question:** If I change or adapt a copyright work, do I still need permission?


**Answer:** Changing or adapting a copyright work does not remove the need for permission, particularly if you are using an important part of the original work. Unauthorised changes to a work can also constitute derogatory treatment under copyright law, which is a separate concern from infringement. If you want to adapt, translate, or create a derivative work, you need permission from the copyright owner.
