# Can I post a copy to myself as proof that it is copyright?

**Question:** Can I post a copy of my work to myself as proof of copyright?


**Answer:** Posting a sealed copy to yourself provides very weak evidence at best. While the postmark establishes a date, there is no verifiable way to prove the contents have not been altered after sealing. Copyright in New Zealand is automatic and does not require registration, but if you need to demonstrate ownership or timing, keeping dated records of your creative process (emails, file metadata, version history) is more reliable.
