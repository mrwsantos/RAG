# If only a small amount of a work is used, is permission still needed?

**Question:** Do I need permission to use a small amount of a copyright work?


**Answer:** Permission may still be required even if you only use a small amount of a copyright work. There is no minimum threshold below which copying is automatically permitted. The only exceptions are where your use fits within a specific fair dealing exception under the Copyright Act, such as research, private study, criticism, review, or news reporting. If none of those exceptions apply, permission is required regardless of how little you copy.
